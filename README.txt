OLANGA BRIDGE EXTENSION — v0.8.3
=================================

INSTALL (Chrome / Edge):
1. Open chrome://extensions/
2. Enable "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select this entire "olanga-extension" folder

USE:
1. Start Olanga: python -m http.server 8080 (in your Downloads folder)
2. Open http://localhost:8080/ollama-quiz-controller.html
3. In Olanga, click "Live Tab" in the Source panel
4. Navigate Chrome to any page you want to control (e.g. google.com)
5. Click the Olanga icon in Chrome toolbar → "Activate on this tab"
6. Go back to Olanga → click "Connect Tab"
7. Type your instruction and let Olanga work

NOTES:
- Works on any webpage including Google, forms, quizzes, etc.
- The extension must be re-activated after navigating to a new page
- Does not work on chrome:// pages or the Chrome Web Store
