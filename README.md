# Olanga
A full-stack agentic browser automation system that uses Gemini API through a multi-step four-phase read/plan/execute/verify architecture to reliably complete goals across dynamic multi-page flows. AI planning and real-time DOM interaction to complete tasks on any webpage, featuring a Chrome extension (via. Manifest v3) with a built-in side panel.
(Smaller sub-tool of My Unrealesed Olanga Computer Assistent Agent)
